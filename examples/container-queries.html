Container Queries vs Viewport Queries

Understanding the difference between modern container queries and traditional viewport queries.

What are Container Queries?

Container queries let components respond to their container size, not the viewport size. This makes components truly reusable!

Viewport queries (media queries) respond to the browser window size.

In dot-grid:

  • .col-md-6 = Container query (default, modern)
  • .col-vp-md-6 = Viewport query (traditional)

When to Use Each

Use Container Queries

  • Reusable components
  • Card grids
  • Widgets that appear in different contexts
  • Component libraries
  • Most modern layouts

Use Viewport Queries

  • Top-level page structure
  • Navigation that changes with screen size
  • Hiding/showing elements globally
  • Typography that scales with viewport
